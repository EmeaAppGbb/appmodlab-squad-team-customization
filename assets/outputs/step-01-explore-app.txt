=== ANALYSIS ===
MedBook is a Go-based healthcare microservices platform with:
- 3 microservices: patient-service, provider-service, appointment-service  
- gRPC + HTTP APIs (Gin framework)
- PostgreSQL database for patient PHI
- Kubernetes deployment manifests
- Existing SQUAD configuration with 7 agents (4 standard + 3 custom healthcare agents)
- Custom ceremonies: weekly HIPAA review, sprint planning, daily standup, retrospective, monthly audit
- Quality gates: pre-commit PHI scan, code review compliance, pre-merge verification
